Introduction
------------

1. cd ~/Desktop/fitbit/

2. npm run run

### Prerequisites:
- Ensure that you have one of the following required operating systems:  
  * [Node.js](https://nodejs.org/en/)

### Installation
- In command line:
* On Mac:
  ```  
    $ npm install
  ```

### Run
- In command line:
* On Mac:
  ```  
    $ npm start
  ```
